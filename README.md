<b>Requirements:</b>
<ul>
<li>virualenv</li>
<li>python-pip</li>

Step to install:
<ul>
<li>clone project</li>
<li>go to project root directory</li>
<li>create a virtual env</li>
<li>inside virualenv run pip install -r requirements.txt</li>
<li>execute python manage.py runserver 0:8000</li>
<li>navigate to localhost:8000</li>
</ul>
